Classes and IDs

Classes and IDs are used to select things via CSS and JavaScript. What's the difference? Well, IDs can only be used once, while classes can be used as many times as you like. You can apply then to an element by using attributes. In order to create an attribut, you need to modify the opening tag for an element: 

			
				<h1>A Header</h1>
				<h1 class="red">A Red Header</h1>

				<p>A Paragraph</p>
				<p class="red">A Red Paragraph</p>
				<p class="red">Another Red Paragraph</p>

				<p id="orange">This is orange!</p>

After creating these elements, you may have noticed that nothing happened to their appearances! This is because you haven't applied any styling to the red class or the orange element. You can select a class in CSS by placing a . before the class name, and you can select an element's id by placing a # before the id: 

			
				.red {
				  color: red;
				}

				#orange {
				  color: orange
				}

This will make the elements have color!

Emmet

The Basics

Emmet is like an "ease of access" element creator. JSFiddle has it built in, along with many popular editors worldwide. You can use it by typing a css selector, then pressing the Tab key. It will expand into an element! For example, if you type h1.red, and press tab, it will turn into <h1 class="red"></h1>.

Other Uses

Emmet also has more uses, like making expanded and nested elements. For example, if you type ol>li*5, it will create an entire ordered list with 5 <li> elements inside of it! Here, the > means "to have inside," and the * means "to have the amount."
