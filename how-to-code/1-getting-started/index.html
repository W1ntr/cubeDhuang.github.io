Welcome to my coding tutorial! You have either come to see what this is about, or to learn how to code. Either way, let's get started!

To get started, we are going to use a site called JSFiddle. JSFiddle does do a few things behind the scenes. For example, it inserts the html, head, and body on its own. We'll talk about those later.

Now, when you open JSFiddle, you will see an HTML section, a CSS section, and a JavaScript section.

JSFiddle provides some pre-styling, but we want to get rid of that. Put this in the CSS section to fix it: 

			
				body { background: white }

Enter this into the HTML section:

			
				<h1>A Header</h1>
				<p>A Paragraph</p>

Press the [Run] button at the top of the screen to render the HTML.

Now, you may be scratching your heads thinking, huh, what does all of this mean? Well, I'm here to explain. The <h1> thing at the beginning means the start of a header tag. There are also <h2>, <h3>, and all the way up to <h6> tags, <h6> being the smallest. Anyways, all of the text after the <h1> tag is a header. That's why A Header is showing large and bold. After that is the </h1> tag. The / at the beginning just means that it is an ending tag. It tells the computer that that is the end of the header, and that all the stuff after is not a header. The <p> tag is the start of a paragraph. It tells the computer that everything after is a paragraph. Then, the </p> tag just ends the paragraph.

That's it! You've just written (and I hope you now understand) some HTML code! Try playing around with the things that you can do will all of these header and paragraph tags. Check out my next article in the series to learn about CSS, and making your page look better.

Also, check out MDN to search up anything you don't know or want to know about HTML, CSS, and JavaScript.
