So far, this is great. We know how to create headers and paragraphs, but what if you want to make lists? Tables? What other things could there be?

Lists

Lists are suprisingly easy: 

			
				<ol>
				  <li>Foo</li>
				  <li>Bar</li>
				  <li>Biz</li>
				</ol>

As always, press run to show the list. Now, we are also nesting HTML elements inside of each other. The indentation for the section is just to help organize our code. The <ol> tag is the start of an ordered list. This makes it so that everything in is is already numbered by default. Each <li> element makes a new element in the list. Our elements are foo, bar, and biz, but you can decide what to put in there. <ul> means unordered list. Try changing both <ol> tags to <ul> tags. Don't forget the slash in the ending tag!

Tables

The Basics

Now that you know how to make lists, we can move on to tables. The <table> element creates a table. Inside of the <table> element, there will be <tr> elements for rows of the table. Inside of the <tr> elements, there will be <th> and <td> elements for Headings and items in the table. Here's an example: 

			
				<table>
				  <tr>
					<th>Element 1 of Row 1</th>
					<th>Element 2 of Row 1</th>
					<th>Element 3 of Row 1</th>
				  </tr>
				  <tr>
					<td>Element 1 of Row 2</td>
					<td>Element 2 of Row 2</td>
					<td>Element 3 of Row 2</td>
				  </tr>
				  <tr>
					<td>Element 1 of Row 3</td>
					<td>Element 2 of Row 3</td>
					<td>Element 3 of Row 3</td>
				  </tr>
				</table>

In this case, you can just copy and paste the code. If you have been doing that all along, then don't do it anymore! Typing the code out by hand will really help you memorize every aspect about it. Later, we will learn how to use Emmet that is built into JSFiddle for convenience.

Organization

Tables can also be labeled and organized with <thead>, <tbody>, and <tfoot> elements. These do not make any visual change, but are good for selection in CSS. We could rewrite the example above with these 3 elements like: 

			
				<table>
				  <thead>
					<tr>
					  <th>Element 1 of Row 1</th>
					  <th>Element 2 of Row 1</th>
					  <th>Element 3 of Row 1</th>
					</tr>
				  </thead>
				  <tbody>
					<tr>
					  <td>Element 1 of Row 1</td>
					  <td>Element 2 of Row 1</td>
					  <td>Element 3 of Row 1</td>
					</tr>
				  </tbody>
				  <tfoot>
					<tr>
					  <td>Element 1 of Row 1</td>
					  <td>Element 2 of Row 1</td>
					  <td>Element 3 of Row 1</td>
					</tr>
				  </tfoot>
				</table>
