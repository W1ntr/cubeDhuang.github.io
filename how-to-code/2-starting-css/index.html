Now that you've learned a bit of HTML, it's time to start learning CSS. Remember that CSS means Cascading Style Sheets, so it's pretty easy to remember what it's for – style!

I am continuing with the code from the previous section, so check that out if you haven't.

In the CSS section, put:

			
				* {
				  font: 200% sans-serif;
				}

Press run to re-render it. This is actually very simple. The `*` means that it should affect everything. In the curly braces is where all of the style for the thing goes. We have indented a bit to show that this is a new part of nested code, which is for global styles. Each of the styles in css must have a semicolon after it.

Right here, we only have the font style. This will change your font to 200% its normal size, and make the font sans-serif. Basically, serif means that the font has lot's of decoration and detail, etc., and sans-serif is more simple and clean. You can also create more specific fonts to get a better look, but we'll just do this for now. This CSS is pretty simple, so let's add some more! ;) 

			
				h1 {
				  font-size: 100px;
				}
				
				p {
				  font-size: 10px;
				}

Press run, and you can see that this has a very exaggerated effect. The heading should be very big, and the paragraph should be very small. You've likely noticed that this is a bit different. The h1 means that we are selecting all <h1> elements on the page. We are changing the font size to 100 pixels! Then, we are doing the same thing with the <p> element, and converting its font size to 10 pixels. The actual font sizes will be 200 pixels and 20 pixels, because of the other CSS that changes everything to be 200% bigger.
